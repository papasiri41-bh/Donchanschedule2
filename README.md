# เว็บบอร์ดตารางนัดหมายสำนักงานปศุสัตว์อำเภอดอนจาน

เว็บแบบ Mobile-first สำหรับดู เพิ่ม และแก้ไขตารางนัดหมายผ่านลิงก์เดียว รองรับมุมมองรายสัปดาห์ รายเดือน และรายปี พร้อมปฏิทินและวันหยุด

## ไฟล์สำคัญ

- `index.html` หน้าเว็บหลัก
- `styles.css` รูปแบบหน้าจอมือถือแนวตั้ง
- `app.js` ระบบปฏิทิน เพิ่ม และแก้ไขนัดหมาย
- `config.js` ค่า Supabase Project URL และ Publishable key
- `supabase-schema.sql` สร้าง/ปรับฐานข้อมูล สิทธิ์ RLS และ Realtime
- `assets/dld-logo.png` ตราสัญลักษณ์กรมปศุสัตว์

## วิธีเปิดใช้งาน

1. เข้า Supabase Dashboard ของโครงการ
2. ไปที่ **SQL Editor > New query**
3. คัดลอกเนื้อหาใน `supabase-schema.sql` วางแล้วกด **Run**
4. เปิด `config.js` ตรวจสอบว่า Project URL และ Publishable key ถูกต้อง
5. อัปโหลดไฟล์ทั้งหมดขึ้น Netlify, Vercel, GitHub Pages หรือโฮสต์เว็บไซต์แบบ Static
6. ส่งลิงก์เว็บไซต์ให้ผู้เกี่ยวข้อง

## ค่า Supabase ที่ใส่ไว้แล้ว

```js
window.SUPABASE_CONFIG = {
  url: "https://ygiapbfqqnhjaestubau.supabase.co",
  anonKey: "sb_publishable_xcdF9DHFPxc97U-HyGmyQg_ftlAH8AS"
};
```

คีย์ที่ขึ้นต้นด้วย `sb_publishable_` เป็นคีย์สำหรับหน้าเว็บ ส่วนคีย์ที่ขึ้นต้นด้วย `sb_secret_` ห้ามใส่ในไฟล์เว็บไซต์และควรเก็บเฉพาะฝั่งเซิร์ฟเวอร์เท่านั้น

## สิทธิ์ที่ตั้งไว้ตามคำขอ

ผู้ที่มีลิงก์สามารถ:

- ดูรายการทั้งหมด
- เพิ่มรายการ
- แก้ไขรายการเดิม

หน้าเว็บไม่มีปุ่มลบ เพื่อช่วยลดความเสี่ยงจากการลบข้อมูลโดยไม่ตั้งใจ

> หมายเหตุด้านความปลอดภัย: การเปิด INSERT และ UPDATE ให้ผู้ใช้สาธารณะหมายความว่าทุกคนที่เข้าถึงลิงก์สามารถแก้ข้อมูลได้จริง จึงไม่ควรใส่ข้อมูลลับหรือข้อมูลส่วนบุคคลที่ละเอียดอ่อน หากภายหลังต้องการจำกัดเฉพาะเจ้าหน้าที่ ควรเพิ่ม Supabase Auth และเปลี่ยน RLS ให้เฉพาะผู้เข้าสู่ระบบ

## การเพิ่มวันหยุด

เปิด Supabase > Table Editor > `holidays` แล้วเพิ่ม:

- `holiday_date` วันที่รูปแบบ `YYYY-MM-DD`
- `name` ชื่อวันหยุด
- `is_official` เลือก `true`

วันหยุดจะปรากฏบนปฏิทินเมื่อรีเฟรชหน้าเว็บ

## การนำขึ้น Netlify แบบง่าย

1. เข้า Netlify และเลือก **Add new site > Deploy manually**
2. ลากโฟลเดอร์นี้ทั้งโฟลเดอร์ไปวาง
3. รอระบบสร้างลิงก์
4. เปิดลิงก์ทดสอบการเพิ่มและแก้ไขนัดหมาย
